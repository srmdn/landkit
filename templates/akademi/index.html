Curriculum

16 weeks, one clear path

Not a random collection of topics. Each module builds on the last, ending in a full-stack portfolio project employers actually care about.

  • How the web works: browsers, HTTP, DNS
  • HTML semantics and document structure
  • CSS layout: Flexbox and Grid from first principles
  • Responsive design and mobile-first methodology
  • Project: Personal portfolio page
  • Variables, types, functions, and scope
  • DOM manipulation and event handling
  • Promises, async/await, and the fetch API
  • Error handling patterns
  • Project: Interactive quiz app
  • Component thinking and reusability
  • useState, useEffect, and custom hooks
  • React Router and multi-page apps
  • State management patterns
  • Project: Full React frontend with live API
  • Node.js runtime and CommonJS modules
  • Express.js: routing, middleware, error handling
  • RESTful API design principles
  • Authentication with JWT
  • Project: Production-ready REST API
  • Relational databases: PostgreSQL fundamentals
  • Joins, indexes, and query optimization
  • ORMs: Prisma in practice
  • When to use NoSQL: MongoDB use cases
  • CI/CD pipelines with GitHub Actions
  • Deploying to cloud: Railway, Vercel, Fly.io
  • Environment management and secrets
  • Monitoring and error tracking basics
  • Capstone: Full-stack SaaS app, production-deployed
