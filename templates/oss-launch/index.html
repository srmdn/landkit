Quickstart

Up and running in 60 seconds

Three steps. No prerequisites beyond Node.js.

1

Install devkit globally

Requires Node.js 18+. Works on macOS, Linux, and Windows.

terminal
$ npm install -g devkit
✓ devkit@2.4.1 installed successfully
2

Initialize your project

Scaffolds the project structure and installs dependencies automatically.

terminal
$ devkit init my-project
✓ Created project my-project
✓ Installed 3 dependencies
→ cd my-project
3

Start developing

Hot reload, error overlay, and instant feedback built in by default.

terminal
$ devkit dev
✓ Ready in 1.2s
✓ Server at http://localhost:3000
✓ Watching for changes...
